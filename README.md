```
Commands:
  timetrack amend ID                      # amend a frame
  timetrack backup                        # dump database to timetrackYYMMDDHHMMSS.sql.gz file
  timetrack chart                         # show chart
  timetrack commit -m, --message=MESSAGE  # commit frame
  timetrack db                            # run database console
  timetrack help [COMMAND]                # Describe available commands or one specific command
  timetrack log                           # query frame log
  timetrack pry                           # start pry console
  timetrack start -c, --client=CLIENT     # start a new time frame
```
